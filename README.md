# Automatic Email

Sistema em Python para automação assistida de composição de e-mails no Outlook.

## Funcionalidades atuais
- Cadastro e edição de clientes
- Preenchimento de Para e CC
- Limpeza da composição ao trocar cliente
- Seleção de anexos
- Geração automática básica de assunto
- Abertura de novo e-mail no Outlook

## Tecnologias
- Python
- PySide6
- SQLAlchemy
- SQLite
- pywin32

## Como executar
1. Criar ambiente virtual
2. Instalar dependências com `pip install -r requirements.txt`
3. Rodar `python -m app.main`
